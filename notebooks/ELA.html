Matrix Algebra

Matrix-Vector Product

Ax = c₁v₁ + c₂v₂ + … + cₙvₙ
A is m×n, x is n×1 → Ax is m×1

Matrix-Matrix Product

If B = [v₁ v₂ … vₚ], then AB = [Av₁ Av₂ … Avₚ]
A is m×n, B is n×p → AB is m×p
(inner dimensions must match, outer dimensions give result size)

Scalar Multiplication

kA = multiply every entry of A by k

Properties

✓ A(BC) = (AB)C  (associative)
✓ A(B+C) = AB + AC  (distributive)
✓ IA = AI = A  (identity)
✗ AB ≠ BA in general  (NOT commutative)
✗ AB = 0 does NOT mean A = 0 or B = 0
✗ AB = AC does NOT mean B = C
Size check first! If inner dimensions don't match, the product is undefined. A 2×3 times a 2×2 = NOT DEFINED.

2C

Linear Independence

The Test

Put vectors as columns → row reduce → check pivots
Pivot in EVERY column → Linearly Independent
Any column WITHOUT a pivot → Linearly Dependent

Instant Decision Rules

! More vectors than dimension of ℝⁿ → always dependent (e.g. 4 vectors in ℝ³)
! Zero vector in the set → always dependent
! Two vectors: dependent ↔ one is a scalar multiple of the other
! Independent ↔ Ax = 0 has ONLY x = 0 (no free variables)

Reading Dependency from RREF

If dependent, the free variable columns tell you the dependency relation. The entries in those columns give the coefficients.

Test: {[1; −1; 4], [3; −5; 7], [−1; 5; h]} — for what h are they dependent?
Form matrix and row reduce:
[1 3 −1; −1 −5 5; 4 7 h]
R2 += R1: [1 3 −1; 0 −2 4; 4 7 h]
R3 −= 4R1: [1 3 −1; 0 −2 4; 0 −5 h+4]
R3 −= (5/2)R2: [1 3 −1; 0 −2 4; 0 0 h+4−10]
Bottom-right entry: h − 6
Dependent when h − 6 = 0 → h = 6
For "find values of h" problems: row reduce until you isolate h, then set the bottom entry to 0 for dependence (or ≠ 0 for independence).
True/False — give a brief justification:

(a) If {v₁, v₂, v₃} is linearly independent in ℝ³, then {v₁, v₂, v₃, v₄} is linearly independent in ℝ³ for any v₄ ∈ ℝ³.

(b) If v₃ is not a linear combination of v₁ and v₂, and all three are in ℝ⁴, then {v₁, v₂, v₃} is a linearly independent set.

(a) FALSE. 4 vectors in ℝ³ are always dependent (more vectors than dimensions). No matter what v₄ is, the set must be dependent. (b) FALSE. v₃ not being a linear combination of v₁, v₂ doesn't guarantee independence of the full set. Counter-example: v₁ could be a multiple of v₂ (making v₁, v₂ dependent). Then {v₁, v₂, v₃} is dependent even though v₃ ∉ Span{v₁, v₂}. e.g. v₁ = [1;0;0;0], v₂ = [2;0;0;0], v₃ = [0;1;0;0]. v₃ is not in Span{v₁,v₂}, but 2v₁ − v₂ = 0 so the set is dependent.
Find all values of h for which the set of vectors is linearly dependent:
{ [2; 1; −3], [−4; 0; 6], [1; h; −2] }
Form matrix [v₁ v₂ v₃] and row reduce: [2 −4 1] [1 0 h] [−3 6 −2] Swap R1 & R2: [1 0 h] [2 −4 1] [−3 6 −2] R2 −= 2R1: [1 0 h; 0 −4 1−2h; −3 6 −2] R3 += 3R1: [1 0 h; 0 −4 1−2h; 0 6 −2+3h] R3 += (3/2)R2: [1 0 h; 0 −4 1−2h; 0 0 −2+3h + (3/2)(1−2h)] Bottom-right = −2 + 3h + 3/2 − 3h = −2 + 3/2 = −1/2 Since −1/2 ≠ 0 for any h, there is always a pivot in every column. The set is linearly INDEPENDENT for ALL values of h. (There is no value of h that makes them dependent.)

Matrices as Linear Transformations

Finding the Standard Matrix

A = [ T(e₁) T(e₂) … T(eₙ) ]
Plug in each standard basis vector, use the outputs as columns.
If A is m×n, then T : ℝⁿ → ℝᵐ

Linearity Test

A function T is a linear transformation if and only if:
T(cu) = cT(u)   AND   T(u + v) = T(u) + T(v)

Equivalently: T(cu + dv) = cT(u) + dT(v)

KEY: T(0) must = 0 for linear transformations!
f(x) = mx + b is NOT linear when b ≠ 0 (fails T(0) = 0)

Composition

First T (matrix A), then S (matrix B):
(S ∘ T)(x) = B(Ax) = (BA)x ← REVERSED ORDER!

Geometric Transformations in ℝ²

TransformationMatrix
Rotation by θ[cos θ −sin θ; sin θ cos θ]
Reflect over x-axis[1 0; 0 −1]
Reflect over y-axis[−1 0; 0 1]
Reflect over y = x[0 1; 1 0]
Scale by k[k 0; 0 k]
Find the standard matrix for T that leaves e₁ unchanged and maps e₂ to e₁ + 3e₂.
T(e₁) = e₁ = [1; 0] ← first column
T(e₂) = e₁ + 3e₂ = [1; 3] ← second column
A = [1 1; 0 3]
Visual: e₁ = [1,0] stays put. e₂ = [0,1] gets sheared to [1,3].
Define f : ℝ → ℝ by f(x) = 3x − 2.

(a) Show that f is not a linear transformation.

(b) If we instead had g(x) = 3x, show that g is a linear transformation.

(a) f(x) = 3x − 2 is NOT linear: Easiest proof: check f(0). f(0) = 3(0) − 2 = −2 ≠ 0 Since f(0) ≠ 0, f cannot be a linear transformation. ✗ Alternative: f(x + y) = 3(x+y) − 2 = 3x + 3y − 2 f(x) + f(y) = (3x−2) + (3y−2) = 3x + 3y − 4 3x+3y−2 ≠ 3x+3y−4, so additivity fails. ✗ (b) g(x) = 3x IS linear: g(cu) = 3(cu) = c(3u) = c·g(u) ✓ (scaling) g(u + v) = 3(u+v) = 3u + 3v = g(u) + g(v) ✓ (addition) Both properties hold, so g is a linear transformation. (Its "matrix" is just the 1×1 matrix [3].)
Find the standard matrix for the linear transformation T : ℝ² → ℝ² that reflects vectors over the x-axis and then scales by a factor of 2. Illustrate by computing T([3; 1]).
Step 1: Find T(e₁) and T(e₂). T(e₁) = T([1;0]): reflect [1;0] over x-axis → [1;0], then scale ×2 → [2; 0] T(e₂) = T([0;1]): reflect [0;1] over x-axis → [0;−1], then scale ×2 → [0; −2] A = [2 0; 0 −2] Or via composition: Reflect x-axis: R = [1 0; 0 −1], Scale: S = [2 0; 0 2] T = S·R (scale AFTER reflect) = [2 0; 0 2]·[1 0; 0 −1] = [2 0; 0 −2] ✓ Compute T([3; 1]): [2 0; 0 −2]·[3; 1] = [6; −2] (The point (3,1) gets reflected to (3,−1), then scaled to (6,−2).)

Invertibility

2×2 Inverse (MEMORIZE)

⎡a b⎤⁻¹ 1 ⎡ d −b⎤ ⎣c d⎦ = ――――――― · ⎣−c a⎦ ad−bc

Invertible ↔ ad − bc ≠ 0
Swap diagonal, negate off-diagonal, divide by determinant

General Inverse: Row Reduce [A | I]

[A | I] row reduce to [I | A⁻¹]
If left side doesn't reduce to I → A is NOT invertible

Invertible Matrix Theorem — All Equivalent

For an n×n matrix A, these are all true or all false together:

✓ A is invertible
✓ RREF of A is I
✓ A has n pivot positions
✓ Columns are linearly independent
✓ Columns span ℝⁿ
✓ Ax = b has a unique solution for every b
✓ Ax = 0 has only x = 0
✓ det(A) ≠ 0
✓ 0 is NOT an eigenvalue
If ONE fails, they ALL fail. Pick whichever is easiest to check.

Key Properties

(A⁻¹)⁻¹ = A (AB)⁻¹ = B⁻¹A⁻¹ ← reversed! Ax = b → x = A⁻¹b Triangular: invertible ↔ all diagonal entries ≠ 0
2×2 Inverse Example:
A = [5 2; −3 1]
det = 5(1) − 2(−3) = 5 + 6 = 11 ≠ 0 → invertible
A⁻¹ = (1/11) · [1 −2; 3 5] = [1/11 −2/11; 3/11 5/11]
Consider A = [4 −6; −2 3].

(a) Is A invertible? Justify using the determinant.

(b) Based on your answer, what can you say about the solutions to Ax = 0?

(a) det(A) = 4(3) − (−6)(−2) = 12 − 12 = 0 A is NOT invertible (det = 0). (b) By the Invertible Matrix Theorem, since A is not invertible: • Ax = 0 has infinitely many solutions (nontrivial solutions exist) • The columns of A are linearly dependent • A does not reduce to the identity • Col(A) ≠ ℝ² (columns don't span all of ℝ²)

Bases & Coordinate Systems

What Is a Basis?

B = {v₁, …, vₘ} is a basis for ℝᵐ if:
1. Linearly independent  AND  2. Spans ℝᵐ

Equivalently: n vectors in ℝⁿ, and their matrix row reduces to I
A basis for ℝⁿ has EXACTLY n vectors.

Coordinate Conversion

Let P_B = [v₁ v₂ … vₙ] (basis vectors as columns)

B-coords → Standard: x = P_B · [x]_B
Standard → B-coords: [x]_B = P_B⁻¹ · x
P_B converts FROM B-coords TO standard. P_B⁻¹ goes the other direction. Don't mix them up!
B-coords to standard:
Basis B = {[3, 1], [−2, 1]}, coordinate vector [x]_B = [2, −1]
x = 2·[3,1] + (−1)·[−2,1] = [6,2] + [2,−1] = [8, 1]
(This means: 2 of the first basis vector, minus 1 of the second.)
Let B = {v₁, v₂} where v₁ = [1; 2] and v₂ = [3; 5].

(a) Verify that B is a basis for ℝ².

(b) Find [x]_B where x = [−1; 1].

(a) P_B = [1 3; 2 5]. det = 1(5) − 3(2) = 5 − 6 = −1 ≠ 0 Since det ≠ 0, the two vectors are independent AND span ℝ² → B is a basis. (b) Solve P_B · [x]_B = x, i.e. [1 3; 2 5] · [c₁; c₂] = [−1; 1] Augmented matrix: [1 3 | −1; 2 5 | 1] R2 −= 2R1: [1 3 | −1; 0 −1 | 3] R2 *= −1: [1 3 | −1; 0 1 | −3] R1 −= 3R2: [1 0 | 8; 0 1 | −3] [x]_B = [8; −3] Check: 8·[1;2] + (−3)·[3;5] = [8;16]+[−9;−15] = [−1; 1] ✓

Determinants

2×2 (MEMORIZE)

det [a b; c d] = ad − bc

3×3 Cofactor Expansion

Pick any row or column (choose one with zeros!). For each entry in that row/column:

Term = (−1)^(i+j) · entry · det(2×2 submatrix)

Sign pattern: [+ − +] [− + −] [+ − +]

Row Operation Effects

OperationEffect on det
Row replacement (Rᵢ + k·Rⱼ → Rᵢ)det UNCHANGED
Swap two rowsdet × (−1)
Scale a row by kdet × k

Properties Cheat Sheet

det(I) = 1 det(AB) = det(A)·det(B) det(A⁻¹) = 1/det(A) det(kA) = kⁿ·det(A) for n×n det(triangular) = product of diagonal entries

det = 0 ↔ NOT invertible ↔ columns dependent det ≠ 0 ↔ invertible
3×3 cofactor expansion (expand along row 1):
A = [2 0 −1; 3 1 2; −2 4 −3]
= +2·det[1 2; 4 −3] − 0·det[3 2; −2 −3] + (−1)·det[3 1; −2 4]
= 2·(−3−8) − 0 + (−1)·(12−(−2))
= 2·(−11) + (−1)·(14) = −22 − 14 = −36
Compute the determinant of the matrix using any method:
A = [1 −2 3; 0 4 −1; 2 0 5]
Method: Cofactor expand along column 1 (has a zero — less work!) Sign pattern col 1: +, −, + = +1·det[4 −1; 0 5] − 0·det[−2 3; 0 5] + 2·det[−2 3; 4 −1] = 1·(4·5 − (−1)·0) − 0 + 2·((−2)(−1) − 3·4) = 1·(20) + 2·(2 − 12) = 20 + 2·(−10) = 20 − 20 = 0 Since det(A) = 0: A is NOT invertible.
3D

Subspaces: Col(A) & Nul(A)

Column Space

Col(A) = span of the columns of A (subspace of ℝᵐ)
Basis: ORIGINAL columns of A at pivot positions
dim Col(A) = rank(A) = # pivots

Use columns of A, NOT columns of the RREF!

Null Space

Nul(A) = {x : Ax = 0} (subspace of ℝⁿ)
Steps: Row reduce A → write parametric form → read off basis vectors
dim Nul(A) = n − rank(A) = # free variables

Rank-Nullity Theorem

rank(A) + dim Nul(A) = # columns
A = [1 3 2; 2 6 5; −1 −3 0]
RREF → [1 3 0; 0 0 1; 0 0 0]
Pivots in columns 1 and 3. rank = 2.
Col(A) basis: {[1;2;−1], [2;5;0]} ← original columns 1 & 3
Free variable: x₂. From RREF: x₁ = −3x₂, x₃ = 0
Nul(A) basis: {[−3; 1; 0]}   dim = 1
Check: 2 + 1 = 3 columns ✓
The RREF of a 3×5 matrix A is:
[1 0 2 0 −1; 0 1 −3 0 4; 0 0 0 1 2]

(a) What is rank(A)? What is dim Nul(A)?

(b) Which columns of the original A form a basis for Col(A)?

(c) Find a basis for Nul(A).

(a) Pivots in columns 1, 2, 4. rank(A) = 3 dim Nul(A) = 5 − 3 = 2 (b) Columns 1, 2, and 4 of the ORIGINAL matrix A. (c) Free variables: x₃ and x₅ From RREF: x₁ = −2x₃ + x₅ x₂ = 3x₃ − 4x₅ x₃ = free x₄ = −2x₅ x₅ = free Set x₃ = 1, x₅ = 0: → [−2; 3; 1; 0; 0] Set x₃ = 0, x₅ = 1: → [1; −4; 0; −2; 1] Nul(A) basis = { [−2; 3; 1; 0; 0], [1; −4; 0; −2; 1] }

Eigenvalues & Eigenvectors

Definition

Av = λv where v ≠ 0
λ = eigenvalue, v = eigenvector (v must be nonzero; λ CAN be 0)

Step 1: Find Eigenvalues

Solve: det(A − λI) = 0 (characteristic equation)

2×2 shortcut for A = [a b; c d]:
λ² − (a+d)λ + (ad−bc) = 0
trace = a+d, det = ad−bc

Triangular matrix: eigenvalues = diagonal entries

Step 2: Find Eigenvectors

For each λ: solve (A − λI)v = 0
Find Nul(A − λI). That's the eigenspace E_λ.
When computing A − λI: subtract λ from ONLY the diagonal entries. If you row-reduce and get I, you made an error.
A = [4 −1; 2 1]
Step 1: λ² − (4+1)λ + (4·1−(−1)·2) = λ² − 5λ + 6 = (λ−2)(λ−3) = 0
Eigenvalues: λ = 2, λ = 3
Step 2 (λ=2): A−2I = [2 −1; 2 −1] → RREF [1 −1/2; 0 0]
x₁ = (1/2)x₂ → v = [1; 2] (set x₂ = 2)
Step 2 (λ=3): A−3I = [1 −1; 2 −2] → RREF [1 −1; 0 0]
x₁ = x₂ → v = [1; 1]
E₂ basis: {[1,2]}, E₃ basis: {[1,1]}
Find all eigenvalues and eigenvectors of:
A = [3 −2; 1 0]
Step 1: Eigenvalues λ² − (3+0)λ + (3·0−(−2)·1) = λ² − 3λ + 2 = (λ−1)(λ−2) = 0 λ = 1 and λ = 2 Step 2: Eigenvectors λ = 1: A − I = [2 −2; 1 −1] → RREF [1 −1; 0 0] x₁ = x₂ → E₁ basis: {[1; 1]} λ = 2: A − 2I = [1 −2; 1 −2] → RREF [1 −2; 0 0] x₁ = 2x₂ → E₂ basis: {[2; 1]}

Diagonalization & Similarity

The Key Formula

A = PDP⁻¹
D = diagonal matrix of eigenvalues
P = [eigenvectors as columns]
Column order in P must match diagonal order in D!

Steps to Diagonalize

1. Find eigenvalues: det(A − λI) = 0 2. Find eigenvectors for each eigenvalue 3. Can you get n linearly independent eigenvectors total? YES → A = PDP⁻¹ NO → not diagonalizable

When Is A Diagonalizable?

✓ n distinct eigenvalues → ALWAYS diagonalizable
✓ For each λ: geometric mult. = algebraic mult. → diagonalizable
✗ Some λ has dim E_λ < algebraic multiplicity → NOT diagonalizable

Powers

Aᵏ = P Dᵏ P⁻¹
Dᵏ = diag(λ₁ᵏ, λ₂ᵏ, …) (just raise each diagonal entry to kth power)

Similarity

A ~ B means A = PBP⁻¹ for some invertible P.
Similar matrices share: eigenvalues, determinant, trace, char. polynomial.
Diagonalize A = [7 2; −4 1]
Eigenvalues: λ² − 8λ + 15 = (λ−3)(λ−5) = 0 → λ = 3, 5
λ=3: A−3I = [4 2; −4 −2] → RREF [1 1/2; 0 0] → v₁ = [−1; 2]
λ=5: A−5I = [2 2; −4 −4] → RREF [1 1; 0 0] → v₂ = [−1; 1]
2 independent eigenvectors ✓
P = [−1 −1; 2 1], D = [3 0; 0 5]
A¹⁰⁰ = P · diag(3¹⁰⁰, 5¹⁰⁰) · P⁻¹
Determine whether A is diagonalizable. If so, find P and D.
A = [5 −6; 3 −4]
Step 1: Eigenvalues λ² − (5+(−4))λ + (5·(−4)−(−6)·3) = λ² − λ + (−20+18) = λ² − λ − 2 = (λ−2)(λ+1) = 0 λ = 2 and λ = −1 (2 distinct → guaranteed diagonalizable!) Step 2: Eigenvectors λ = 2: A−2I = [3 −6; 3 −6] → RREF [1 −2; 0 0] x₁ = 2x₂ → v₁ = [2; 1] λ = −1: A+I = [6 −6; 3 −3] → RREF [1 −1; 0 0] x₁ = x₂ → v₂ = [1; 1] P = [2 1; 1 1], D = [2 0; 0 −1] A = PDP⁻¹
